## 🔧 Featured Add-Ons For Unity and Blender 

### [VRChat PhysBone to Magica Cloth 2 Converter (PBToMC2) 🦴](https://github.com/NyakumaEmi/VRC-PhysBone-To-Magica-Cloth-2) (Currently Private Repo)
**PBToMC2** is a **Unity** automation utility designed designed to quickly and efficiently migrate VRChat PhysBone setups to Magica Cloth 2. It automates the tedious process of cloning avatars, mapping bone hierarchies, converting physics parameters, and re-linking colliders. 

This is useful for converting VRChat avatars to other platforms, such as Warudo for 3D VTubing.
### [TEXTOPO 🎨](https://github.com/NyakumaEmi/TEXTOPO) (Currently Private Repo)
**TEXTOPO** is a **Unity** automation utility designed to automate the merging of four independent textures into a single high-resolution texture atlas. 

It saves time by not having to manually align images in image editing software like Clip Studio Paint or Photoshop.
### [Multi-Mesh UV De-Triangulate △→□](https://github.com/NyakumaEmi/Multi-Mesh-UV-De-Triangulate) (Currently Private Repo)
**Multi-Mesh UV Guided De-Triangulate** is a **Blender** 5.0 add-on designed to de-triangulate an armature's meshes by using the UV Map of the original quad-based meshes as a guide to merge triangles back into quads. This works even if the mesh has been scaled, moved, sculpted, or deformed to fit a new character (clothing). Requires that the vertex count + UV map be the same between paired meshes. 

This tool was specifically built in mind for VRChat avatar creators and 3D outfitters who need to undo or reverse the triangulation that happens when assets are processed through game engines like Unity and are exported as FBX files. This add-on matches together by name the child meshes of both a source armature and a target armature.
